## Dataset

**Source:** [Data Science Salaries 2023 — Kaggle](https://www.kaggle.com/datasets/arnabchaki/data-science-salaries-2023)

**File:** `ds_salaries.csv`  
**Size:** ~3,700 rows | 11 columns  
**Key columns:** `job_title`, `experience_level`, `employment_type`, `salary_in_usd`, `remote_ratio`, `company_size`, `company_location`

Download and place in `/data/` before running the notebook.

## Setup & Execution

### 1. Install dependencies
```bash
pip install pandas numpy scikit-learn jupyter
```

### 2. Run the notebook
```bash
cd notebook
jupyter notebook job_market_pipeline.ipynb
```
Run all cells top to bottom. Outputs are saved automatically.

### 3. Deploy frontend
- Push `/frontend/` to GitHub
- Connect to [Vercel](https://vercel.com) → deploy as static site
- OR open `index.html` directly in browser (demo mode with built-in data)

The notebook writes `dashboard_data.json` into both `/outputs/` and `/frontend/`, so the deployed static site can load the real data without a backend.

## ML Models

| Model | Type | Metric |
|---|---|---|
| Gradient Boosting Regressor | Regression (Salary) | MAE ~$35.7k, R²~0.47 |
| K-Means (k=5) | Clustering (Job Groups) | Silhouette ≈ 0.43 |

## Features

- **Salary Predictor** — Input role + experience + company size → predicted salary range
- **Cluster Viewer** — PCA scatter plot of 500 role samples across 5 job clusters
- **Market Analytics** — Median salary by experience, top-10 job titles bar chart
- **Similar Jobs** — Roles in the same predicted cluster shown post-prediction
- **Zero backend** — All inference uses pre-exported JSON; runs on Vercel free tier
